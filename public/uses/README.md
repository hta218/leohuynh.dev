# /uses images

Drop product photos / logos here and reference them from `src/pages/uses.astro`
via the `image` field, e.g. `image: '/uses/macbook.webp'`.

When an item has no `image`, the page falls back to the favicon of its `href`,
then to a letter monogram. Prefer square-ish images, ~256px, `.webp` or `.png`.
