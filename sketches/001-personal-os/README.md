## Variant: Personal OS / Dev Console

### Design stance
Make the homepage feel like Leo's living personal dashboard: writing, activity, projects, media, stats, and search in one high-signal interface.

### Key choices
- Layout: dashboard hero + cards + activity rail.
- Typography: sharp sans with monospace operational labels.
- Color: dark-first, violet/cyan/green accents.
- Interaction: command palette mockup via `⌘K`.

### Trade-offs
- Strong at: personality, showing many content types, feeling modern.
- Weak at: can become busy if every widget ships at once.

### Best for
A personal site that is more than a blog: portfolio, notes, activity, and live status.
