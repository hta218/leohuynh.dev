## Variant: Minimal Coder Light

### Design stance
A quiet light-mode homepage with subtle coder details: terminal chips, contribution stats, and a living activity rail without feeling like a dashboard SaaS.

### Key choices
- Layout: airy two-column hero, latest writing, sticky status rail.
- Typography: clean sans body with restrained monospace labels.
- Color: warm white, ink, soft green/blue accents.
- Interaction: activity filter buttons.

### Trade-offs
- Strong at: minimal feel, readable, still personal/coder.
- Weak at: less visually dramatic than previous sketches.

### Best for
The likely v4 direction if the goal is simple, light, personal, and long-term maintainable.
