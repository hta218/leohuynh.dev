## Variant: Fullscreen Code Studio

### Design stance
Make the homepage feel like a clean browser-based code workspace: full-width, light-mode, readable, but clearly programmer-native.

### Key choices
- Layout: full-width IDE shell with file tree, central README/home content, right runtime widgets.
- Typography: clean sans for prose, monospace for system/status/code surfaces.
- Color: off-white editor canvas, muted borders, green/blue code accents.
- Interaction: switch file tabs between README, activity, and posts.

### Trade-offs
- Strong at: coder vibe, full-screen presence, strong personal-system feel.
- Weak at: needs careful responsive design so it does not feel like a fake IDE gimmick.

### Best for
A v4 homepage that should feel “programmer” immediately while staying light and minimal.
