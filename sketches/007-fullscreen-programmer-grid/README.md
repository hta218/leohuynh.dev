## Variant: Fullscreen Programmer Grid

### Design stance
A full-width personal dashboard that feels like a programmer's daily control surface: command strip, contribution heatmap, writing list, and activity stream.

### Key choices
- Layout: full-bleed asymmetric grid with large writing canvas and live data cards.
- Typography: bold editorial headline plus monospace system labels.
- Color: light canvas with restrained black/green/blue accents and tiny pixel details.
- Interaction: toggle activity stream between all, code, and life.

### Trade-offs
- Strong at: more wow than the minimalist versions, still light and coder-oriented.
- Weak at: denser homepage; needs discipline to avoid dashboard creep.

### Best for
A v4 direction that should feel more alive and programmer-specific than a plain blog.
