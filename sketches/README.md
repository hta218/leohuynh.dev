# leohuynh.dev v4 design sketches

Throwaway HTML mockups for choosing the v4 design direction before the Astro/Bun rebuild.

## Variants

1. `001-personal-os/` — dark dashboard / personal operating system.
2. `002-editorial-engineer/` — reading-first editorial layout.
3. `003-brutalist-lab/` — high-personality developer lab/notebook.
4. `004-minimal-coder-light/` — light-mode minimal coder homepage with activity rail.
5. `005-plain-code-journal/` — very minimal code-editor/journal style.

Open locally:

```bash
open sketches/001-personal-os/index.html
open sketches/002-editorial-engineer/index.html
open sketches/003-brutalist-lab/index.html
open sketches/004-minimal-coder-light/index.html
open sketches/005-plain-code-journal/index.html
```

These are disposable visual spikes, not production implementation.
